## Born from a real inbox

fast-classifier was born from a live session that organized a real 6,551-email inbox: **3,849 newsletters swept in one run**, **5,573 emails filed into 14 labels**, and **87% rule coverage** reached by iterating recon reports instead of training anything. The one-off scripts from that session are preserved read-only in [`reference/`](https://github.com/ahmadatallah/fast-classifier/tree/main/reference); this project is those scripts hardened into a typed, tested library with a CLI and an MCP server. Every [core concept](concepts/01-mcp-handshake/) shipped from that session.
